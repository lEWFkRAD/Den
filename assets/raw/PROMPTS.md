# Den — Asset Generation Prompts

Use these prompts with your preferred AI image generator.

## Characters (5 sheets = ~20 characters)
```
dark moody tactical RPG character sheet, 4 characters standing, full body,
clean background, designed for sprite conversion, pixel art style,
dark fantasy armor and weapons, muted tones with accent colors
```

## Kips (8-10 creatures)
```
ancient elemental war construct creature, dark fantasy mechanical familiar,
glowing runes, isolated character, clean background,
creature design sheet, multiple angles, game asset
```

Variants:
- blood kip: crimson veins, organic metal, dripping red energy
- electric kip: copper coils, crackling sparks, yellow-blue glow
- void kip: obsidian shell, purple rifts, reality distortion
- light kip: golden filigree, radiant wings, warm white glow
- plant kip: bark armor, vine tendrils, moss and flowers
- dark kip: shadow form, violet eyes, smoke trail
- ice kip: crystalline shell, frost patterns, pale blue
- god kip: all elements, prismatic, ancient and massive

## Terrain Props (10-15 props)
```
dark fantasy battlefield props, top down tactical RPG assets,
ruined statues, broken siege weapons, ancient stone altars,
isolated objects, clean background, game asset sheet
```

## Weapon/Artifact Icons (128x128 export)
```
dark fantasy artifact icon, isolated weapon, clean background,
game UI icon, detailed metalwork, magical glow
```

Icon subjects:
- void crystal orb
- bone crown
- blood ritual dagger
- storm conductor staff
- shadow lance
- flame greatsword
- ice tome
- nature bow
- divine shield
- cursed axe

## Terrain Tiles (tileable, 6-8 types)
```
top down tactical RPG terrain tile, tileable texture, game map asset,
dark fantasy style, muted natural colors
```

Tile types needed:
- grass (base)
- stone/cobble
- ruins/cracked stone
- water (animated if possible)
- snow/ice
- sand/desert
- lava (glowing)
- dark void ground

## Sprite Conversion Workflow
1. Crop character from sheet
2. Resize to 64x64 or 96x96
3. Clean edges (remove background artifacts)
4. Export as PNG with transparency
5. Place in: assets/portraits/{name}_small.png (for grid)
6. Full portrait: assets/portraits/{name}.png (for UI panel)
